# 矿山无人车毫米波雷达目标追踪与感知融合项目
## 项目职责
主导单/多毫米波雷达感知模块的架构开发、算法适配与工程落地，核心解决矿山扬尘场景下激光雷达失效导致的目标漏检、无人车异常刹停问题，搭建毫米波雷达安全兜底的感知融合追踪链路，保障无人车全天候连续作业的安全性。

## 核心工作与技术方案
1.  **追踪算法选型与工程化落地**
    - 基于工业界落地优先原则，选定SORT算法搭建多目标追踪链路，核心采用卡尔曼滤波做目标状态预测、匈牙利算法做帧间目标关联，保证追踪链路的低时延、高稳定性与可解释性；
    - 针对大陆ARS408毫米波雷达无加速度输出的传感器特性，采用CV匀速模型，定义状态向量为`[x, y, vx, vy]`（世界坐标xy、xy方向速度），避免无观测支撑的状态噪声，适配矿山场景目标运动特性；
    - 基于目标平面距离、已存活帧数构建匈牙利匹配代价矩阵，摒弃易受环境干扰的速度、RCS维度，最大化匹配链路的稳定性，减少矿卡会车时的ID互换与关联错误。

2.  **目标级感知融合链路搭建**
    - 设计**激光雷达主检测、毫米波雷达安全兜底**的目标级融合架构，采用双源并行输出、硬逻辑仲裁的融合策略：世界坐标系下，毫米波目标中心落在激光3D检测框范围内时，优先输出激光检测结果；激光漏检导致目标未被覆盖时，平滑切换至毫米波目标兜底输出，全程无目标跳变与断链；
    - 搭建前置多级过滤链路，针对矿山场景毫米波虚警、多径效应、RCS值波动问题，设计分档RCS阈值过滤、栅格高度过滤、静态背景过滤、孤立点过滤等策略，仅将高置信度目标输入追踪模块，从根源减少追踪跳变与虚警误跟。

3.  **全链路鲁棒性与安全边界优化**
    - 明确项目ODD边界，核心覆盖矿山道路上的动态工程车辆与行驶路径内的静止故障车，基于毫米波相对速度检测能力，保证静止故障矿卡的正常检出，无安全漏洞；
    - 搭建标准化坐标转换链路，完成单雷达坐标→左前激光雷达坐标→世界坐标的两级空间对齐，设置标定有效性前置校验机制，每次上线前完成标定精度校验，避免标定误差传递导致的目标偏移；
    - 优化目标生存周期与续帧逻辑，设置3帧目标生存窗口，解决单帧掉帧、目标短暂遮挡导致的追踪断链问题，保障扬尘场景下目标的持续追踪能力。

4.  **多毫米波雷达扩展适配**
    - 完成从单正前毫米波雷达到车前、左前、右前多毫米波雷达的架构扩展，搭建多雷达目标时空同步与合并逻辑，覆盖车辆全视角感知范围，解决侧向会车、近距离目标的漏检问题。

## 量化成果
- 核心性能：扬尘场景下，毫米波雷达FOV内目标持续追踪率≥95%，完美承接激光雷达失效时的安全兜底需求；
- 误检优化：多级过滤策略将毫米波虚警率下降85%以上，异常刹停频率下降80%，解决了土墙、排土场多径反射导致的误检问题；
- 实时性：全链路单帧平均处理时延≤12ms，极端场景超时阈值设置为100ms，满足车端10Hz主控制周期的实时性要求，预留充足安全冗余；
- 工程落地：完成全链路车端实时部署，支撑矿山无人车7×24小时连续稳定作业，无感知模块导致的安全事故与业务中断。
